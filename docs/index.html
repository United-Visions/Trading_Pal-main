Trading Pal Documentation

Capabilities

Trading Pal offers a comprehensive suite of tools for algorithmic trading, including:

  • Backtesting engine
  • Data server with real-time market data
  • Interactive charting library
  • Dual Broker System (Alpaca, Oanda) for comprehensive market coverage (Forex, Crypto, Stocks)
  • Natural Language Trading
  • User-friendly interface

Getting Started

To get started with Trading Pal, follow these steps:

  1. Install the necessary dependencies: pip install -r requirements.txt
  2. Configure your broker credentials in user_config.py
  3. Run the application: python main.py

Data Server

The data server provides real-time market data via a REST API. See the API documentation for details.

Charts Display Logic

Trading Pal uses a client-side charting library to display market data. The library is highly customizable and allows for various chart types and indicators.

Backtesting Logic

The backtesting engine uses historical market data to simulate trading strategies. The results are presented in a clear and concise format, allowing for easy analysis and optimization.
